Intel Map

css, style, fullscreen, window, margin, padding, scrollbar

This is a map of geo-codable stories submitted to news.intelmap.com, a Hacker News clone built in Arc language (source code here). The map is built on OpenLayers, and the submitted URL's are geo-coded using GeoNames' RSS2GeoRSS API service. It was done for the sake of tinkering; send feedback to bradley.taft.smith@gmail.com.
